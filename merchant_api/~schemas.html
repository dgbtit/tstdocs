Merchant API

​Schemas

​StoreInfo

  • name_vistring

    Store name in Vietnamese

    Example: Store Hồ Chí Minh
  • name_enstring

    Store name in English

    Example: Store HCM

​SKUInfo

  • skustring

    SKU code

    Example: SKU001
  • quantityinteger

    Quantity of SKU. Default = 1

    Example: 1
  • priceinteger

    Price of SKU in bill

    Example: 100000

​VoucherConditions

  • start_datestring

    Promo start date (YYYY-MM-DD)

    Example: 2025-01-10
  • exclude_specific_datestring[]

    Promo non-effective dates (YYYY-MM-DD)

  • exclude_recurring_daystring[]

    Promo non-effective day of week

  • redeemable_skusstring[]

    List of redeemable SKUs of the voucher code. For voucher type = conditional, bill number must contain at least 1 redeemable SKU of the voucher.

​VoucherRedemptions

  • redeem_sku_codesobject[]

    Contains redeemed SKU information of the voucher (for voucher type is conditional and support sku)

  • redemption_valueinteger

    Actual redemption value of voucher type = conditional

    Example: 10000
  • used_storeobject

    Store information

  • used_datestring

    Date voucher marked as used in case the voucher has been redeemed. Format (YYYY-MM-DD HH:MM:SS)

    Example: 2025-01-10 12:00:00
  • bill_numberstring

    Bill number for which voucher used/reserved

    Example: BILL001

​VoucherDetail

  • codestring

    Voucher code

    Example: 070898022379
  • valueinteger | null

    Value of voucher

    Example: 100000
  • stateinteger | null

    State of voucher

    Example: 1
  • voucher_typestring

    Voucher type, standard or conditional

    Example: conditional
  • expiry_datestring

    Expiry date of voucher (YYYY-MM-DD)

    Example: 2025-12-26
  • cancel_datestring

    Date cancel voucher (YYYY-MM-DD)

  • conditionsobject

    Include information involve with voucher type is conditional

  • redemptionsobject

    Include information related to the use of the voucher (all types)
