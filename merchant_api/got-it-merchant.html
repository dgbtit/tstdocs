​Check multiple vouchers are valid or not

POST
https://openapi-stg.gotit.vn
/merchant/v6.0/checkmultiple

Check multiple vouchers are valid or not

​Check multiple vouchers are valid or not › Request Body

  • pinstring

    Store pin

    Example: 1212
  • codesstring[]

    Array of 10-16 characters Got It voucher codes

    Example: ["070898022379","7132616187"]
  • bill_numberstring

    Bill number will apply vouchers

    Example: BILL001
  • total_billinteger

    Total bill amount

    Example: 1000000
  • skip_reserved_when_mark_usedboolean

    When true the system will execute the flow without reserve

    Example:
  • skus_infoobject[]

    SKU information in bill_number

​Check multiple vouchers are valid or not › Responses

    • successboolean
      Example:
    • return_codestring | null

      Result code if failed. In case of successful request: value is null

    • message_enstring

      Message notification in English

      Example: Valid voucher
    • message_vistring

      Message notification in Vietnamese

      Example: Voucher hợp lệ
    • dataobject[]

      Detail items of voucher, if result is failed, response will return the first voucher code which is invalid
